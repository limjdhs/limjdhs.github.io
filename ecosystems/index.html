Ecosystem Simulation

An ecosystem is a complex community of living organisms that interact with their physical environment. This simulation will measures the carrying capacity of the ecosystem, which is the maximum number of organisms that can be supported indefinitely. See what changes when you revise the food web by clicking on the organisms, and/or modify the resources by adjusting the sliders.

How can humans harm an ecosystem?

Habitat loss can come from human development and industrialization, as well as environmentally harmful practices like clear cutting. Water quality can drop due to issues such as acid rain or contaminants from improper sewage disposal. Air pollution can be caused by events like emissions from manufacturing activities and the burning of fossil fuels.

We start our code with a for loop, which will repeat our code for each organism in the list of allOrganisms.

Then we move to an if statement, which checks if the organism is active on the food web.

If the organism is active, we'll start the next conditional.


If the organism is a carnivore, the initial value is set to 500.

Otherwise if it is an omnivore, the initial value is 250.

If it is an herbivore, it is 166.66.

And finally, producers are 150.


If the organism is not active, we'll set the value to 0.

We adjust the values of the organism graph using the same computational concepts as setting the intial values: for loops and conditionals.

But what ecological ideas are included? We'll look closely at resources (the amount of habitat, amount of water, and air quality) and prey (if there are enough organisms in the food web to support predators).
